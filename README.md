# Prisma2d2

```
Visualize a Prisma schema as a d2 diagram

Usage: prisma2d2 [OPTIONS] [INPUT_FILE]

Arguments:
  [INPUT_FILE]  Parse the Prisma schema from a file. Defaults to stdin

Options:
  -o, --output-file <OUTPUT_FILE>  Write the d2 diagram to a file. Defaults to stdout
  -h, --help                       Print help
  -V, --version                    Print version
```
